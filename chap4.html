Chapter 4 : Internet et Web

1) Introduction :

Le web est une partie des contenus circulant sur un support de communication générale called internet, et il n'est qu'une de ses applications qui nous permet de consulter des pages accessibles sur des sites avec un navigateur.

Principe WWW :

  • Accéder à des documents textuels.
  • Situé sur des machines accessibles par Internet.
  • Relié entre eux par un mécanisme de lien 《 hypertexte 》.
  • Actuellement : service des ressources.
  • De différentes natures.

    • Uniform Resource Locator (URL) :

      -URL est le nom unique d'une page accédé par un lien hypertexte.

    2) Aspect techniques du Web :

  • URL.
  • HTTP.
  • HTML.

    • 3) Page web et site Web :

    Page web :

    Une page web est un document multimédia, programmé en HTML, et stocké dans un fichier html.
    Elle est interprétée par un navigateur, accédé sur internet par URL.

    Site web :

    Un site Web est une liaison entre des pages Web par des liens hypertexte autour d'une thématique commune, Elles partagent la même entité et accessibles par des URL et des pages d'accueil communes sur un même serveur.
    Structuré comme un disque local, permet de définir des chemins relatifs.

    4) Création d'une page Web HTML :

    - Un document HTML est :

  • Un fichier texte, structuré par des balises.
  • Interpréter dans un navigateur.
  • Créé dans un éditeur de pages Web.

    • Les principes de langage HTML :

    Les balises : les 4 balises de bases sont :

    HTML:Délimite le début et la fin du document.

    HEAD:Entête du document.

    TITLE:Titre du document.

    BODY:Corps du document.

    Les balises secondaires sont :

    hn: Titre de niveau n 1 à 6.

    p: Paragraphe.

    • Une balise est structuré comme suit :

    <Balise> ...</balise>
